# WinTerrEdit
GUI based terraria inventory / player editer.    
Made for the latest version (1.4.0.4/5)    

Supports editing inventory items:    
  ---Item type    
  ---Item quantity    
  ---Item prefix (large, mythical, etc)
  
Supports editing stats:    
  ---Current health    
  ---Maximum health    
  ---Current mana    
  ---Maximum mana    
  
Supports editing colours:    
  ---Hair colour    
  ---Skin colour    
  ---eye colour    
  ---Shirt colour    
  ---Undershirt colour    
  ---Pants colour    
  ---Shoe colour
